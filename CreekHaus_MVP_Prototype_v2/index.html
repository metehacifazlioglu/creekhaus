MVP Prototype

Clickable developer-handoff prototype for the CreekHaus staff platform — employee app, manager app, and web admin. Built to the approved visual direction, with the refinements from the finalization guide and the latest 1:1 applied.
Version 2 · Developer Handoff · Click any screen to open it full-size
How to read this package. These screens are real HTML/CSS (not images), so developers can inspect exact colors, spacing, type, and behavior — the prototype is the spec. The interactions are live: tap checklist items on My Shift, the mic on Daily/Incident, the category chips, and the Guides search. Start with the Documentation at the bottom — the Change Log and Developer Implementation Notes explain every decision. The original PNG mockups are preserved in _Original PNG Prototypes (reference).
App Flow Diagrams  → The whole system on one page — all three surfaces and how a worker's tap becomes manager and executive visibility. Best starting point for the all-in-one view.
